# Java Servlet
#### Java Servletとは、Webサーバ上(バックグランド)で動く、Javaのことです。
Webサーバ上で動かすためには、アプリケーションサーバ(APサーバー)が必要です。よく使われるのは、TomactというAPサーバです。
```
+------Client-------+       +----------------Server----------------+
|                   |       |                                      |
|    +---------+    |       |    +---------+                       |
|    | browser |----+-------+--->| Servlet |-----------+           |
|    +---------+⇦---+--+    |    +---------+⇦--------+ |           |
|                   |  |    |         |              | |           |
+-------------------+  |    |         ↓              | ↓           |
                       |    |    +---------+        +----------+   |
                       +----+----|   JSP   |        |    DB    |   |
                            |    +---------+        +----------+   |
                            |                                      |
                            +--------------------------------------+
```
